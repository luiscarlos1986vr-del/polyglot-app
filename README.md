## **Contexto del Problema** 🌍  
Una startup de comercio electrónico, **Global-Gadgets**, quiere expandirse a mercados como **Japón, Alemania y Brasil**. El desafío es generar contenido de marketing **no solo traducido, sino culturalmente adaptado** para cada región.  

**Polyglot** automatiza este proceso conectándose a diferentes LLMs y generando:  
✔ **Posts para redes sociales** (ej. Twitter/X).  
✔ **Emails promocionales**.  
✔ **Eslóganes publicitarios**.  

---

## **Requisitos del Proyecto** ✅  

### **1. Conectividad Multi-LLM**  
- Soporta al menos **3 APIs de LLMs** (DeepSeek, Mistral, Gemini).  
- Permite seleccionar o comparar modelos.  

### **2. Generación de Contenido**  
- Entrada: Descripción de un producto en español/inglés.  
- Salida:  
  - Post para redes sociales (~280 caracteres).  
  - Email promocional (~150 palabras).  
  - Tres eslóganes publicitarios.  

### **3. Localización**  
- Genera contenido en **japonés, alemán y portugués (Brasil)**.  
- Adapta el **tono y referencias culturales** (ej: formal para Alemania, enérgico para Brasil).  

### **4. Comparación de LLMs (Opcional)**  
- Ejecuta el mismo *prompt* en diferentes LLMs y muestra resultados comparativos.
